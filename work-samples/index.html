North Cloud (Go)

github.com/jonesrussell/north-cloud · northcloud.one

Real-time content intelligence pipeline: crawl sources, store and classify documents, publish events to downstream consumers. Described upstream as microservices with separate PostgreSQL databases per service area, Elasticsearch for content indices, and Redis Pub/Sub for routing to subscribers (for example a social publisher consumer).

Representative stack from the public tree and docs:

  • Go services (source manager, crawler, classifier, publisher, pipeline, search, auth, index manager, and others)
  • PostgreSQL and PL/pgSQL across services
  • Redis Pub/Sub and Elasticsearch
  • Docker Compose for dev and prod compose files, Task for local workflows, tests, and lint
  • JWT auth service, Nginx edge routing, MinIO where configured
  • Observability: Grafana, Loki, Grafana Alloy, Pyroscope (per repository architecture notes)
  • Vue and TypeScript frontends for dashboard and search (collaboration-friendly boundary with API services)

Development practice: active work runs on Spec-Driven Development via Spec Kitty. Features and tasks live in a kanban with git-worktree isolation, agent behavior is constrained by SKILL.md schemas, and every task produces a reviewable artifact (PR, doc, or fix) that goes through normal code review.

Waaseyaa (PHP) and Minoo (live app)

Framework: github.com/waaseyaa/framework · App: github.com/waaseyaa/minoo · Site: minoo.live

Waaseyaa is an entity-first PHP stack built on Symfony components (alpha-stage framework; public description stresses CMS-style, data-driven applications and a layered architecture). It is the base for production applications in the Waaseyaa org.

Minoo is the flagship public deployment: an Indigenous knowledge platform that aggregates structured cultural and language content. The published README states it is powered by Waaseyaa CMS and North Cloud ingestion, exposes JSON:API style routes and JSON Schema, ships PHPUnit unit and integration suites, and deploys to minoo.live via GitHub Actions and PHP Deployer with nginx and TLS.

Overlap with PHP-heavy backend roles: modern PHP 8.x, Composer, Symfony ecosystem patterns, API-first surfaces, automated tests gating deploy, and production operations documentation (nginx, php-fpm, releases).

Development practice: active work uses Spec-Driven Development via Spec Kitty (kanban + git-worktree isolation, SKILL.md schemas constraining agent behavior, reviewable artifacts per task).
